## tsm: Terminal Stock Monitor
 Monitor your portfolio from the command line!
 
 ![Example](https://github.com/kkirchhoff01/tsm/blob/master/doc/tsm.png)

## Using tsm:
To run, simply type:

    ./tsm

To add/remove stocks use +/-
To quit use q or Ctrl-C

# Dependancies
- sqlite3
- requests
- curses
